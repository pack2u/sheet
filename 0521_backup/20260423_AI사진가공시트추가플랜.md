# 20260423 AI 사진 가공 시트 추가 플랜

## 1. 목표 (Goal Description)
상세페이지 제작 시 필요한 일회용기 제품 사진을 AI를 활용하여 다양한 연출 컷으로 가공하고, 이를 확인한 후 특정 Google Drive 폴더에 일괄 저장할 수 있는 Google Sheets 기반의 자동화 시스템을 구축합니다.

## 2. 주요 기능 및 요구사항
- **다양한 연출 컷 생성**: 원본 일회용기 사진을 바탕으로 다음의 이미지들을 AI로 생성합니다.
  - 배경 적용 (예: 고급스러운 식탁, 피크닉 등)
  - 다양한 보는 각도 (Top-down, 45도 측면 등)
  - 음식이 담긴 연출 사진 (Food Plating)
  - (치수 기입은 이번 단계에서 제외)
- **미리보기 및 선택 기능**: 생성된 여러 장의 사진을 시트 내에서 또는 사이드바를 통해 미리보고 원하는 사진만 선택할 수 있습니다.
- **특정 폴더 저장**: 선택 완료된 사진들을 지정된 Google Drive 폴더에 고해상도로 자동 저장합니다.
- **디자인 원칙**: 아이콘 생성 및 UI 구성 시 칼라풀한 아이콘은 배제하고, 단색의 고급스럽고 심플한 아이콘만 사용합니다 (사용자 규칙 적용).

## 3. 사용자 검토 필요 (User Review Required)

> [!IMPORTANT]
> **AI 이미지 생성 모델 확정**
> 사용자 요청에 따라 **Gemini** 모델을 활용하여 이미지 가공을 진행합니다.

> [!NOTE]
> **치수 기입(Dimensioning) 예외 처리**
> 사용자 요청에 따라 치수 기입 이미지는 이번 개발 단계에서 제외합니다.

## 4. 오픈 퀘스천 (Open Questions)

> [!NOTE]
> 1. **원본 이미지 업로드 방식 확정**: 사이드바 UI를 통해 사용자의 PC에 있는 사진 파일을 직접 업로드하는 방식을 사용합니다.
> 2. **저장 폴더 구조 및 파일명 확정**: 
>    - **폴더**: 고정된 기본 폴더 아래 이미 생성되어 있는 '상품명' 하위 폴더에 자동 저장합니다.
>    - **파일명**: `{해당폴더명}_AI_{넘버}` 형식으로 저장됩니다. (예: `샐러드용기_AI_1.jpg`)

## 5. 시스템 아키텍처 및 구현 제안 (Proposed Changes)

### 5.1. Google Sheets & Apps Script UI
- **사이드바(Sidebar) UI 추가**: 시트 우측에 [AI 사진 가공] 사이드바를 띄워 직관적인 옵션 선택(배경, 각도, 음식 종류 등) 및 생성 버튼 제공. 단색의 심플하고 고급스러운 아이콘 배치.
- **결과 미리보기 패널**: 생성된 4~5장의 이미지를 사이드바 혹은 시트 내 특정 시트에 썸네일 형태로 렌더링하여 체크박스로 선택할 수 있도록 구현.

### 5.2. Apps Script (GAS) 백엔드 로직
- `ImageGenerator.gs`: 선택된 AI API와 통신하여 프롬프트를 전송하고 생성된 이미지 URL 또는 Base64 데이터를 받아오는 스크립트.
- `DriveManager.gs`: Google Drive API를 활용하여 선택된 이미지를 지정된 폴더에 저장하는 로직.

## 6. 검증 계획 (Verification Plan)
- **단위 테스트**: 샘플 일회용기 사진 1장을 시스템에 넣고, 3가지 옵션(배경, 각도, 음식)에 대해 정상적으로 프롬프트가 구성되고 Gemini API가 호출되는지 확인.
- **UI 및 저장 테스트**: 사이드바에서 생성된 이미지가 올바르게 렌더링되는지 확인하고, 최종 선택한 파일이 지정된 구글 드라이브 폴더에 `.jpg` 또는 `.png` 포맷으로 정확히 저장되는지 수동 검증.
