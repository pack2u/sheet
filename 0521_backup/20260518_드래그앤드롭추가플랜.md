# 송장 매칭 시스템 드래그 앤 드롭 업로드 기능 추가

## 작업 일자
- 2026년 05월 18일

## 작업 내용
엑셀 파일 업로드 방식을 더 직관적이고 편리하게 개선하기 위해, **'드래그 앤 드롭(Drag & Drop)'** 기능을 추가했습니다.

## 상세 수정 사항
- **대상 파일**: `invoiceMatchSidebar.html`
- **변경 내용**:
  - 기존의 단조로운 파일 선택창을 시각적으로 뚜렷한 '드롭 존(Drop Zone)' 박스 형태로 디자인을 변경했습니다.
  - 마우스로 엑셀 파일을 끌어서 영역 위에 올리면 시각적인 피드백(배경색 변화)을 제공하도록 드래그 오버 이벤트를 추가했습니다.
  - 드롭(놓기) 시 즉시 파일이 인식되고, 내장된 SheetJS 라이브러리가 동작하여 텍스트 창에 내용을 변환해 주도록 로직을 연결했습니다.
  - 클릭해서 탐색기를 열어 파일을 선택하는 기존 방식도 드롭 존 클릭 시 동일하게 유지되도록 구현했습니다.

## 개선 효과
사용자는 엑셀 파일을 직접 드래그해서 사이드바의 텍스트창 위 영역에 놓기만 하면 즉시 매칭 데이터를 불러올 수 있어, 작업 속도와 편의성이 크게 향상됩니다.
