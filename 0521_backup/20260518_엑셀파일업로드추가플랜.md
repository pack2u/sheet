# 송장 매칭 시스템 엑셀 파일 업로드 기능 추가 완료

## 작업 일자
- 2026년 05월 18일

## 작업 내용
카카오 송장 매칭 시스템에서 엑셀 파일(.xlsx, .csv 등)을 직접 업로드하여 바로 매칭할 수 있도록 기능을 추가했습니다.

## 상세 수정 사항
- **대상 파일**: `invoiceMatchSidebar.html`
- **구현 방식**: 
  - 브라우저 내에서 엑셀 파일을 읽을 수 있는 `SheetJS(xlsx.js)` 라이브러리를 추가했습니다.
  - 기존 텍스트 입력창 위에 `<input type="file">` 업로드 버튼을 배치했습니다.
  - 사용자가 파일을 업로드하면 시스템이 엑셀 첫 번째 시트의 데이터를 자동으로 텍스트(탭 구분)로 변환하여 텍스트창에 채워줍니다.
  - 이후 기존 로직과 동일하게 [분석] 버튼을 통해 빠르고 정확하게 매칭을 진행할 수 있습니다.

## 주의(권장) 사항
- 업로드하는 엑셀 파일은 '이름'과 '송장번호' 열만 남겨두고 업로드하시는 것이 매칭 인식률을 가장 높이는 방법입니다.
- 불필요한 데이터가 섞여 있으면 매칭에 실패하는 항목이 생길 수 있습니다.
