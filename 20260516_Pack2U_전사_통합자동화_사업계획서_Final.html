03
As-Is Analysis & Necessity

왜 지금 당장 '자동화'가 필요한가? (위기 진단)

물리적 한계: 3,500개 SKU의 압박과 인력 번아웃

현재 3,500가지의 막대한 제품군(SKU)을 관리 중이며, 단기 내 500여 가지의 신제품이 추가될 예정입니다.
이는 이미 인간의 뇌와 엑셀 수작업으로 감당할 수 있는 한계점(Threshold)을 완전히 돌파한 상태입니다.

❌ 수동 트래킹 불가 (단가/재고 연동 실패)
❌ 치명적 휴먼 에러 기하급수적 증가 (오배송/로스)
❌ 전화/카톡 폭주로 인한 CS 인력 극심한 과부하

물류: 모래성 같은 엑셀

VLOOKUP 의존. 열(Column) 1개만 어긋나도 4,000개 제품의 송장이 연쇄 붕괴. 매일 폭탄 돌리기식 수동 운영.

콘텐츠: 수작업 병목

신제품 500개 입고 시 썸네일 누끼부터 HTML, 전산 등록까지 전 과정을 인간이 수작업. 출시 지연의 주범.

세일즈: 수수료 종속

외부 마켓 의존도 90% 이상. 자체 데이터 확보 불가로 4,000개 상품에 대한 단골 재구매 유도(Push) 불가능.

04
Platform Strategy

구글 시트 의존성 탈피와 Web App 기반 고도화

현재: Google Sheets 환경의 한계

  • 데이터 한계 (Quota): 일일 API 호출 수 및 셀 갯수 제한(1,000만 개)으로 인해 대규모 스케일업 불가.
  • 퍼포먼스 저하: 수천 건의 송장을 한 번에 매핑할 때, 스크립트 실행 시간(Timeout 6분) 초과 위험.
  • 휴먼 에러 노출: 누구나 실수로 셀을 지우거나 필터를 잘못 걸면 전체 데이터가 꼬이는 원초적인 UI 문제.

미래: 전용 Web App 구축 시 장점

  • 무한한 확장성 (Scalability): 클라우드 데이터베이스(PostgreSQL 등)를 도입하여 1억 건의 주문도 지연 없이 0.1초 만에 검색/처리 가능.
  • 강력한 권한 제어 (Auth): 협력사나 직원이 볼 수 있는 화면과 메뉴를 DB 레벨에서 완벽히 통제 가능 (보안사고 Zero).
  • 모바일/PC 반응형 UI: 비좁은 엑셀 화면이 아닌 직관적이고 쾌적한 전용 대시보드 인터페이스 제공.
  • 실시간 양방향 연동 (WebSockets): 이카운트 ERP 업데이트 시, 웹 화면에서 즉각적으로 알림 팝업 및 데이터 자동 갱신.

"구글 시트는 훌륭한 초기 프로토타입(MVP) 도구였으나, 월 매출 수십억 단위의 엔터프라이즈 환경에서는 반드시 독립적인 Web App(클라우드 서버+DB)으로 이관되어야 합니다."

07
Core Backend

코어 물류망 전면 개편 (엑셀 파괴)

"열(Column) 의존성 탈피 및 무결성 확보"

기존 시트의 수식(VLOOKUP) 의존도를 0%로 만듭니다. 모든 데이터는 JSON 객체(Key-Value 쌍) 형태로 변환되어 백엔드에서 처리되므로, 사원들이 실수로 시트의 행렬을 지워도 시스템은 붕괴하지 않습니다.

  • 데이터 모델링: 비정형 엑셀 데이터를 정규화된 RDBMS 스키마 형태로 파싱하여 메모리 적재
  • 무인 자동 매핑: 협력업체 상품명과 이카운트 품목코드 간의 Levenshtein 유사도 매핑
  • 보안 격리: API 인증키 완전 은닉 및 IP 기반 접근 통제(ACL)

이카운트 OAPI 다이렉트 브리지 및 큐 아키텍처

엑셀 다운로드 후 수동 업로드 방식은 전면 폐기됩니다. "다이렉트 브리지(REST API)"를 통해 실시간 트랜잭션을 발생시킵니다.

  • API 규격: 이카운트 SaveOrder, SaveSale 엔드포인트 직접 호출
  • 세션 유지: 로그인 Session ID를 캐싱하여 불필요한 인증 요청 최소화
  • 장애 복원력: 통신 오류 발생 시 메시지를 데드레터 큐(DLQ)에 보관 후 지수 백오프 자동 재시도

단일 DB 기반으로 관리자가 대시보드에서 원가 인상률(예: +15%)을 입력하는 즉시 전 채널에 실시간 동기화됩니다.

  • B2C 마켓 방어: 사방넷 API를 통해 50여 개 외부 마켓 판매가가 1분 내 일괄 상향 조정되어 역마진 원천 차단
  • B2B 파트너 자동 고지: 파트너 전용 웹앱에 "원자재가 상승 단가 변동" 팝업 자동 노출 및 실시간 단가표 반영
  • 수요 예측 발주: 글로벌 위기 징후 감지 시 AI가 안전 재고를 선제 확보하도록 자동 발주 알림

"이 시스템은 인건비 절감을 넘어, 예측 불가능한 글로벌 시장 충격(Price Volatility)으로부터 회사의 마진을 수호하는 가장 강력하고 민첩한 방패입니다."
