사업 목표

전 구간 완전 자동화

주문 수집, 재고 할당, 송장 매핑, 출고, 이카운트 전표 처리까지 인간의 개입이 필요 없는 Zero-Touch 데이터 파이프라인 완성.

에러/로그 관제 고도화

중앙 집중식 로깅 체계 구축. API 에러 및 장애 발생 시 Slack/Telegram 기반 즉각적인 모니터링 알림 전송.

확장성 및 유연성

AI 사진 가공(Gemini), 자동 마진 계산기, 현장 모바일 바코드 스캐너 등 신기능 확장을 완벽히 수용하는 견고한 아키텍처.

보안 및 권한 강화

스크립트 하드코딩 제거 및 API 키 외부화. 운영자 및 파트너별 정교한 권한 제어로 시스템 탈취 및 오염 원천 차단.

현행 구조 한계 진단

보안/권한 취약성

코드 내 API Key 평문 노출. 전역 권한 허용으로 인한 외부 파트너의 의도치 않은 시스템 조작 리스크 존재.

스크립트 과부하 및 의존도

특정 파일에 기능이 과밀되어 수정 시 연쇄 장애 발생 우려. 절대 좌표(열 인덱스) 의존성 높음.

차기 시스템 마스터플랜

1

코어 시스템 안정화 및 보안 (1~2개월)

비밀키 `PropertiesService` 이관, 역할별 디렉토리 분리, 중앙 집중식 에러 표준 로그 시트 도입.

2

핵심 파이프라인 지능화 (3~4개월)

예외 건 자동 재처리 큐(Queue), 상태 머신 표준화, 엑셀 업로드 불필요한 이카운트 다이렉트 전송 API 모듈 완성.

3

스마트 관제 및 기능 확장 (5~6개월)

AI 이미지 가공(Gemini), 자동 마진계산기, 모바일 PDA 연동, Web 기반 중앙 관제 대시보드 구축.

기대 효과 및 로드맵

  • 운영비용 90% 절감: 수동 송장 매핑 및 엑셀 업로드 제거
  • 에러율 Zero화: 백오프 알고리즘 기반 자동 재시도 큐
  • 실시간 모니터링: Slack/Telegram 알림으로 MTTR 대폭 감소
  • SaaS형 플랫폼: UUID 기반 웹 DB 설계로 향후 스케일업 완벽 대비
