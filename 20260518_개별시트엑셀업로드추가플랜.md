# 개별 협력업체 시트용 엑셀 업로드/드래그 앤 드롭 기능 반영

## 작업 일자
- 2026년 05월 18일

## 작업 배경
- 각 협력업체에서 직접 자신의 시트에 엑셀 송장을 업로드하여 매칭할 수 있도록 해달라는 요청이 있었습니다.
- 기존 중앙 관리용 시스템에만 적용했던 최신 UI(드래그 앤 드롭 엑셀 업로드)를 개별 시트에 배포되는 스크립트에도 동일하게 이식해야 했습니다.

## 작업 내용
- **대상 파일**: `_partnerInvoiceScript.gs` (`_pt_buildInvoiceMatchHtml_` 함수)
- **수정 사항**:
  - 각 협력업체 시트에 원격으로 설치되는 인라인 HTML 소스코드 문자열 안에 `SheetJS` 라이브러리와 엑셀 변환 로직, 드래그 앤 드롭 UI를 모두 추가했습니다.
  - 이제 배포 스크립트를 실행하면 각 업체의 시트에도 중앙 시스템과 똑같은 강력한 업로드 기능이 탑재된 사이드바가 덮어씌워집니다.

## 배포 가이드
각 업체의 시트에 이 기능을 적용하시려면 메인 시트에서 아래 메뉴를 한 번 실행해 주세요:
**[🛠️ AS / 진단] > [📋 시트·탭 관리] > [사이드바 구형 스크립트 배포 (전체 업체)]**
위 메뉴를 실행하시면 모든 업체의 시트에 최신 버전의 송장 매칭 사이드바가 일괄 설치(업데이트)됩니다.
