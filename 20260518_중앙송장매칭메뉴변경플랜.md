# 송장 매칭 시스템 중앙 관리 메뉴 개편 플랜

## 작업 일자
- 2026년 05월 18일

## 작업 배경
- 기존에는 각 협력업체 시트마다 스크립트를 일일이 설치(배포)하여 개별적으로 송장 매칭을 진행해야 했기 때문에 관리가 매우 불편했습니다.
- 사실 이미 중앙 허브 시트에서 모든 협력업체를 선택하여 원격으로 매칭을 쏴주는 "중앙 관리용 사이드바"가 존재했으나, 위치가 AS/복구 메뉴 깊숙이 숨어 있어 활용하기 어려웠습니다.

## 작업 내용
1. **메인 메뉴로 이동 및 접근성 강화 (`_partnerMenu.gs`)**
   - 기존 `[🛠️ AS / 진단]` 안쪽에 있던 송장 매칭 사이드바 실행 버튼을 메인 업무 메뉴인 **`[📦 대리발송 발주시스템(New)]`**으로 꺼냈습니다.
   - 이름도 **"📬 [중앙 관리용] 송장 매칭 시스템 (카카오/엑셀)"**으로 변경하여 용도를 명확히 했습니다.
   
2. **구형 스크립트 배포 기능 분리**
   - 일일이 시트에 스크립트를 넣던 기존 방식의 메뉴 이름은 "사이드바 구형 스크립트 배포"로 변경하고 AS 메뉴에 남겨두어, 더 이상 불필요한 개별 설치 작업을 하지 않도록 유도했습니다.

## 개선 효과
이제 메인 관리 시트(허브) 하나만 띄워둔 상태에서, 위 메뉴를 클릭해 사이드바를 열고 **업체 목록 드롭다운에서 원하는 업체를 선택**한 뒤 송장 데이터(텍스트 또는 엑셀)를 넣으면 해당 업체의 시트로 원격 매칭이 즉시 적용됩니다. 업체마다 돌아다닐 필요가 없어 업무 효율이 극대화됩니다.
