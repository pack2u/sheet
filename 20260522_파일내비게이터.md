# Pack2U 상품정보sheet 프로젝트 파일 내비게이터

왼쪽 파일 탐색기에서 한글 폴더명이 깨져 보이는 문제를 해결하기 위해, 한글이 깨지지 않고 클릭 한 번으로 파일을 열 수 있는 내비게이터를 생성했습니다.
원하는 파일 이름을 클릭하시면 즉시 해당 에디터 창이 열립니다.
