# 🔍 상품검색 사이드바 기능 정리서 (20260610)

## 1. 기능 개요

협력업체 시트(대리판매/대리공급)에서 **상품명 또는 이카운트코드로 검색**하여, 검색 결과를 클릭하면 **발주 및 송장조회 탭의 C열(이카운트코드)**에 자동 입력해주는 사이드바 기능입니다.

### 사용 흐름
1. 협력업체 시트 열기 → 상단 메뉴 `🔍 상품 검색` 클릭
2. `상품명으로 코드 검색` 클릭 → 우측 사이드바 열림
3. 검색창에 품목명 또는 코드 입력 → 300ms 디바운스 후 자동 검색
4. 결과 목록에서 클릭 → **발주 및 송장조회 탭의 C열 마지막 데이터 다음 행**에 자동 입력
5. 토스트 메시지로 "✅ {코드} 입력 완료" 표시

## 3. 배포되는 함수 상세

### 3-1. `onOpen()` (Code.gs)

```javascript
function onOpen() {
  // 1) 공지 팝업 (Y1이 비어있거나 (, #으로 시작하면 건너뜀)
  try {
    var msg = SpreadsheetApp.getActiveSpreadsheet().getSheets()[0].getRange('Y1').getValue();
    if (msg && msg.charAt(0) !== '(' && msg.charAt(0) !== '#') {
      // 🔔 공지사항 모달 표시
    }
  } catch(e) {}

  // 2) 🔍 상품 검색 메뉴 등록 (무조건 실행)
  try {
    SpreadsheetApp.getUi()
      .createMenu('🔍 상품 검색')
      .addItem('상품명으로 코드 검색', 'openProductSearchSidebar')
      .addToUi();
  } catch(eMenu2) {}
}
```

### 3-2. `searchProductByName(query)` (ProductSearch.gs)

단가조회(뷰어) 탭에서 상품을 검색하는 서버 함수:
- 단가조회 또는 뷰어 탭을 자동 탐색
- Row 4부터 데이터 읽기 (A:상태, C:코드, D:품목명, G:단가)
- 품목명 또는 코드에 검색어 포함 시 결과에 추가 (최대 50건)

### 3-3. `insertEcountCode(code)` (ProductSearch.gs)

검색 결과 클릭 시 C열에 코드를 입력하는 함수:
- 대상 시트 찾기: `발주 및 송장조회` → `전용양식` → 현재 시트 순서로 탐색
- **C열 기준** 마지막 데이터 행 찾기 (ARRAYFORMULA spill 무시)
- 마지막 데이터 다음 행 C열에 코드 입력 + 커서 이동

### 3-4. `_getProductSearchHtml_()` (ProductSearch.gs)

사이드바 UI를 구성하는 인라인 HTML 생성 함수:
- 검색 입력창: 300ms 디바운스 실시간 검색
- 결과 카드: 코드(파란 볼드), 품목명, 상태(품절/단종 빨간색), 단가
- 클릭 → `insertEcountCode(code)` 호출
- 토스트: 성공/실패 메시지 2초간 표시

## 4. 배포 방식

### 배포 경로
```
허브 시트 메뉴
  └─ 💼 협력업체 관리 → 🛠️ AS / 진단 → 🔧 복구 도구
       ├─ 📝 스크립트 재설치          → createViewerNoticeScript_()
       └─ ⚙️ 스크립트 캐시 초기화     → adminClearSingleVendorScriptId()
```

### 캐시 관리
- PropertiesService 키: `VIEWER_BOUND_SCRIPT_ID_{시트ID}`
- 재사용: 저장된 ID가 유효하면 코드만 PUT으로 덮어씀
- 재생성: 404/410/403/401 에러 시 캐시 삭제 후 새 프로젝트 생성

## 5. 발견된 버그 및 수정 이력 (20260610)

| # | 버그 | 원인 | 수정 |
|---|------|------|------|
| 1 | 전용양식 시트에서 메뉴 미표시 | `onOpen()`에서 공지 없으면 `return;`으로 함수 전체 종료 | `if/else` 분기로 변경 |
| 2 | 200행부터 코드 입력됨 | `sheet.getLastRow()`가 ARRAYFORMULA spill 포함 | C열만 역순 스캔으로 마지막 데이터 행 탐지 |
| 3 | 403/401 에러 시 영구 배포 실패 | 해당 에러코드에서 캐시 미삭제 | 403/401도 캐시 삭제 대상에 추가 |
