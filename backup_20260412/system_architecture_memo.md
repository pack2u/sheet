# 📦 팩투유 (FactoYou) 위탁 배송 및 쇼핑몰 송장 분배 시스템 아키텍처

이 문서는 AI 어시스턴트 방장(Antigravity)이 다음 작업 세션(내일)에도 팩투유 시스템의 복잡한 톱니바퀴 구조를 잊지 않고 정확하게 코드를 반영하기 위해 데이터베이스에 영구 저장하는 시스템 요약서입니다.

## 1. 데이터 파이프라인 (ETL 구조)
1. **[수집]**: 팩투유(FactoYou) 솔루션을 통해 스마트스토어, 쿠팡, 11번가 등 모든 오픈마켓의 주문 데이터를 마스터 구글 시트 엑셀로 한 번에 당겨옵니다.
2. **[분리]**: 우리 창고에서 나가는 물건이 아닌, '외부 대리발송(위탁발송) 업체'로 던져야 하는 주문들만 `업체별대리발송` 탭으로 분리 필터링합니다.
3. **[송장 수합 - 📍현재 우리 작업 타겟]**: 여러 위탁발송 업체들이 제각기 다른 택배사와 다른 방식(카톡 텍스트, 엑셀 붙여넣기, 이미지 사진 등)으로 보내오는 중구난방 송장번호들을 이 구글 시트에 매핑해야 합니다.
   * **과거 발생했던 대참사**: AI 스크립트 적용을 위해 임의로 **A열을 새로 추가했더니**, 모든 기둥이 우측으로 한 칸씩 밀려나면서 아래 4번 역분배와 5번 이카운트 파이프라인이 연쇄 붕괴됨.
   * **현재 AI 백엔드 튜닝 상태**: 엑셀 헤더의 단어(품목, 수량, 옵션, 단가, 상태 등)를 스스로 읽어 블랙리스트 열로 배제하고, 순수하게 이름/전화번호만 정밀 타겟팅하여 빈 행의 송장번호를 낚아채는 **[V4 광역 안전 엔진]**이 개발 완료 및 이식 대기 중!
4. **[역분배]**: 송장 매핑이 완료되면, 구글 시트 하단에 구분되어 있는 **쇼핑몰별 전용 탭들**(스마트스토어 탭, 쿠팡 탭 등)에 걸려있는 엑셀 함수(`VLOOKUP` 등)가 송장번호를 쭉쭉 빨아들여 각 쇼핑몰 고객에게 배송 출발 알림을 쏩니다.
5. **[ERP 전송]**: 이카운트 ERP 전송용 탭(`판매전표입력`, `품목등록,변경`, `매입전표` 등)은 A열부터 끝열까지 위치 값(Index)이 1mm의 오차도 없이 뼛속까지 하드코딩 되어 있으므로, 시트 전체의 열 추가/삭제는 묻지도 따지지도 않고 **절대 금지 구역**입니다.

---

## 2. 내일 (다음 세션) 즉시 실행할 작전 지침서
* **절대 규칙**: 기존 구글 시트의 **열(Column) 구조를 단 하나도 새로 추가/수정하지 않는다.** (함수 밀림, 팩투유 오류, 이카운트 API 붕괴 완벽 차단)
* **사장님께서 하실 일**: 
   - 원상 복구된 원본 시트에서 **"기존 구조상 원래 송장번호가 들어가야만 하는 빈 기둥 열의 알파벳(예: H열, K열 등)"**만 찾아주시면 됩니다.
* **AI가 할 일**: 
   - 사장님이 좌표를 찍어주시면, `autoTracking.gs` 시스템 변수 `trackCol` 숫자 1개만 해당 기둥 위치로 수정하여 V4 엔진의 타겟 과녁을 재조준합니다. (예: K열이라면 trackCol = 10 으로 단 1초 만에 세팅 변경 후 영구 가동)

## 3. 차기 개발 마스터플랜 (2026년 4월 논의 확정안)
본 시스템은 단순 연동을 넘어선 '스마트 관제 탑' 수준으로 진화 중이며, 다음 3가지 핵심 축을 기준으로 순차 개발을 대기 중입니다.

1. **중앙 에러 및 로그 관제 모니터 (Error Logging)**
   - 백그라운드 서버에서 일어나는 모든 연동 내역과 API 에러를 전용 `[시스템_로그]` 시트에 시간순 기록.
   - 치명적인 에러 시 텔레그램/슬랙 등을 통해 사장님 스마트폰으로 즉시 경고 푸시 알림 발송.

2. **메인 시트 다이렉트 수정 (엑셀 업로드 파괴 엔진)**
   - **문제**: 이카운트에 단가/재고 수정 시 양식에 맞춰 복사하거나 엑셀로 다운받아 업로드해야 했음.
   - **해결**: `상품정보(최적화사본)` 등 메인 시트에서 숫자를 고치고 [다이렉트 전송] 버튼을 누르면, AI 스크립트가 '변경된 값'과 '품목코드'만 안전하게 뽑아내어 이카운트 서버에 무설정 API로 꽂아 넣음. (재고 수정 시, 기존 값과 비교하여 차액만큼 '재고조정전표'를 자동 발행하는 마법 로직 포함)
   - 탭(Tab) 감지 기능으로 엉뚱한 탭에서 실행되지 않도록 메뉴 권한을 분리 차단.

3. **스마트 알림창 & 창고 현장용 모바일 스캐너**
   - **안전재고 경고 봇**: 품목 재고가 임계치 밑으로 떨어지거나 매입가/출고가 마진율이 무너지면 즉시 발주 알림을 보냄.
   - **모바일 PDA 바코드 연동**: PC 엑셀 대신, 스마트폰 카메라로 상품 바코드를 찍어 즉시 단가를 수정하고 이카운트 전표를 전송하는 현장 중심 웹앱(Web-App) 구현.
